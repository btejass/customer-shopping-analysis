# customer-shopping-analysis
End-to-end Data Analytics project using Python, SQL, and Power BI

 📊 Customer Shopping Behavior Analysis

🔹 Project Overview

This is an end-to-end data analytics project where raw customer shopping data is cleaned, analyzed, and visualized to generate business insights.

---

🔹 Tech Stack

* Python (Pandas, NumPy)
* MySQL
* Power BI
* SQLAlchemy

---

🔹 Project Workflow

1. Data Cleaning (Python)

* Handled missing values using group median
* Created new features (age group, purchase frequency)
* Standardized column names

2. Data Analysis (SQL)

* Revenue analysis by gender
* Discount impact analysis
* Customer segmentation (New, Returning, Loyal)
* Top products identification

 3. Data Visualization (Power BI)

* Interactive dashboard with KPIs
* Product and customer insights
* Filters for dynamic analysis

---

 🔹 Key Insights

* Loyal customers contribute the most revenue
* Discounts increase purchase frequency
* Subscribers spend more on average
* Certain products dominate sales

---

 🔹 Project Files

* Python Notebook → `/python`
* SQL Scripts → `/sql`
* Power BI Dashboard → `/powerbi`
* Presentation → `/presentation`

---

🔹 Conclusion

This project demonstrates an end-to-end data analytics pipeline:
Python → SQL → Power BI
